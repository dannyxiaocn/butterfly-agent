# `entity/nutshell_dev/skills/nutshell`

The repo-development skill for nutshell itself.

## What It Contains

`SKILL.md` documents the repo layout, workflow expectations, testing habits, and conventions for adding providers, tools, and other runtime features.

## How To Use This Part

Edit this skill when the repo structure or maintenance workflow changes and you want future `nutshell_dev` sessions to load the updated instructions on demand.

## How It Contributes To The Whole System

This skill keeps repo-specific knowledge out of the always-on system prompt while remaining available whenever development work starts.
